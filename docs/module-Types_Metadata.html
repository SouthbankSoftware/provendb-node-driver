Metadata

Source:

Properties:
Name Type Description
hash string

Hash value for the document. The hash includes metadata attributes but (of course) not the hash itself
minVersion NumberLong

The database version in which this version of the document first became visible
maxVersion NumberLong

The database version in which this version of the document last became visible
forgotten string | null

If true, the contents of this document have been forgotten
_id ObjectId

The "ProvenDB" _id - this is the same for all versions of a document
mongoId ObjectId

The actual MongoDB "_id" of the document. Unlike the provenDB _id , this value changes every time a document is updated.

Metadata that is appended to each document.

Type:
  • object

"_provendb_metadata": {
   "_id": ObjectId("5d1bdf3df945589f847ee9e3"),

   "maxVersion": NumberLong("9223372036854775807")
}
